AI Vision Assist – Object Detection & Voice Feedback

AI Vision Assist is an intelligent tool built for visually impaired individuals. It captures real-time video from a webcam and uses the powerful YOLOv8x (You Only Look Once) object detection model to identify everyday objects in the environment. Upon detection, the system converts the object labels into speech using pyttsx3, allowing users to "hear what the camera sees".

Built with Python, this project leverages the Ultralytics YOLOv8x model for object detection, OpenCV for camera handling and image processing, and pyttsx3 for offline text-to-speech functionality. The goal is to assist users in identifying obstacles or objects like people, bottles, chairs, doors, etc., in both indoor and outdoor environments.

Technologies used: Python, OpenCV, Ultralytics YOLOv8x, pyttsx3, NumPy, Tkinter (for optional GUI).

Benefits: Improves spatial awareness for visually impaired users, can work offline (no internet needed), and provides real-time feedback.

Limitations: Performance depends on camera quality and lighting conditions. Currently identifies common objects only; no custom training yet. Voice announcements may overlap if too many objects are detected simultaneously.
